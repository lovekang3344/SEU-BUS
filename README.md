# 东南大学校园接驳车实时查询系统

这是一个为东南大学师生设计的校园接驳车时刻查询系统，帮助大家方便快捷地查询校内各条接驳车线路的发车时间，提升出行效率。（基于2025年10月20日起执行的最新时刻表）

## 功能特色

### 🕒 实时班车查询
- 根据当前时间智能显示接下来的班车信息
- 实时计算并显示距离发车时间（等待时间）
- 支持工作日和节假日不同的时刻表
- 默认上车地点为兰台

### 🔍 反向查询功能
- 可根据目的地查询所有开往该地点的班车
- 显示从不同站点出发的时间信息
- 便于规划到达特定地点的最佳乘车方案

### 📅 完整时刻表浏览
- 提供所有线路的完整时刻表
- 支持按站点筛选查看
- 可同时查看工作日和节假日时刻表

### 📱 手机端提醒功能
- 支持手机端设置班车提醒
- 声音提醒：使用Web Audio API播放提示音
- 震动提醒：使用Vibration API实现手机震动
- 弹窗提醒：显示详细的班车信息
- 提醒倒计时：实时显示距离提醒触发的时间

### 💻 PC端提醒功能
- 通过浏览器通知功能提醒用户班车即将发车
- 可自定义提醒时间（1-15分钟）
- 支持设置和取消提醒

### 🤖 自动日期检测
- 使用timor.tech API智能判断当前日期类型
- 自动识别工作日和节假日
- 包含法定节假日和调休工作日数据

### 📱 响应式设计
- 完美适配手机、平板和电脑等不同设备
- 移动端友好的界面布局
- 触控友好的交互设计

### 🎨 现代化界面
- 使用Vue 3框架构建
- 使用TailwindCSS进行样式设计
- 简洁清新的用户界面设计
- 直观的信息展示方式
- 便捷的操作流程

## 使用说明

1. **自动日期检测**：系统会自动检测当前日期类型（工作日/节假日）
2. **选择查询模式**：
   - **实时查询**：选择上车地点，查看即将发车的班车
   - **反向查询**：选择目的地，查看开往该地点的所有班车
   - **完整时刻表**：浏览所有线路的完整时刻信息
3. **设置提醒**：
   - 在实时查询和反向查询模式下，可以为班车设置提醒
   - 支持手机端（声音+震动）和PC端（浏览器通知）
   - 可自定义提前提醒时间（1-15分钟）
   - 可随时取消已设置的提醒

## 技术栈

- **前端框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **样式框架**：TailwindCSS
- **API服务**：timor.tech（节假日数据）
- **Web API**：
  - Web Audio API（声音提醒）
  - Vibration API（震动提醒）
  - Notification API（PC端通知）

## 技术特点

- 现代化前端架构，使用Vue 3 Composition API
- 响应式设计，完美适配各种设备
- 组件化开发，代码结构清晰
- 使用TailwindCSS实现高效样式开发
- 集成第三方API提供智能日期判断
- 纯前端实现，无需后端服务器
- 数据存储在JSON文件中，便于维护更新
